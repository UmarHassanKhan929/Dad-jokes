# Dad Jokes

A lightweight, dependency-free dad joke generator that fetches random jokes from the icanhazdadjoke API.

## Features

- Zero dependencies (no npm, no frameworks)
- Uses native `fetch` API
- XSS-safe DOM rendering
- Responsive Bootstrap layout

## API

Uses [icanhazdadjoke](https://icanhazdadjoke.com/) - no API key required.
